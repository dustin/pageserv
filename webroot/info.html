Dustin's Pager Server Info

This is just a little bit of info about my pager server.

Operation

The following protocols exist in this version of pageserv:
  • The original pageserv protocol
  • An SNPP server.
  • A Web server.
It logs via syslog and currently is configurable via the configuration section log. Currently, the only configuration option supported in section log is facility. This specifies the log facility in which the message will be sent to syslog. The default log facility is local7.
