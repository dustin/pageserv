Dustin's Pager Server Info

This is just a little bit of info about my pager server.

Operation

The normal pager server itself runs on port 1029. It accepts the following commands:
  • epage
  • mash
  • farkle *
  • depth
  • quit
It logs via syslog and currently isn't configurable for that, so it just logs to whatever you have set in the logfile. Mine is local7.info for most things.
